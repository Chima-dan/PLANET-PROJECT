# 🌍 Planet Project

A backend Node.js application that processes NASA Kepler exoplanet datasets to identify potentially habitable planets. The application reads large CSV datasets, filters planets based on scientific criteria, and exposes the results through a RESTful API. This project demonstrates data processing, backend architecture, REST API development, and efficient handling of external datasets.

## ✨ Features

- Parse NASA Kepler CSV datasets
- Identify habitable planets
- RESTful API
- Efficient CSV stream processing
- Modular backend architecture
- Error handling
